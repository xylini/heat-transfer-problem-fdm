# Heat Transfer Problem using FDM
## Requirements
* Python 3
* matplotlib
* ifort

## How to run it
Open terminal and type:
```
./run_and_show.py
```

It may take a while, so be patient or before running delete
line 7 ```call(['./run_all.sh'])``` and then run py script.
